# ChatWindow

Use: Vue + Axios

Construction:

```shell
.
├── README.md
├── data
│   └── chat.json
├── images
│   └── default.png
├── index.html
└── src
    ├── css
    │   └── style.css
    └── js
        ├── axios.min.js
        ├── main.js
        └── vue.min.js
```

Components: `messageBox` in `main.js`

> If the URL of chat data isn't accessible, use the code in main.js: 69 and disable web security of Chrome or use Edge instead
